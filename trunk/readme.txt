== Description ==
Easy to use Wordpress plugin to accept paypal payment for a service or product or donation in one click. Can be used in the sidebar, posts and pages.

For information and updates, please visit:
http://www.tipsandtricks-hq.com/?p=120

Usage:
There are two ways you can use this plugin:

   1. Add the trigger text <!-- wp_paypal_payment --> to a post or page
   2. Call the function from a template file: <?php echo Paypal_payment_accept(); ?>


== Installation ==

1. Unzip and Upload the folder 'WP-accept-paypal-payment' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings and configure the options eg. your email, Subject text etc.
4. Add the trigger text <!-- wp_paypal_payment --> to a post or page where u want it to appear.
